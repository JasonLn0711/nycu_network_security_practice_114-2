# HW01 Instructions: Reverse Engineering Foundations

## Goal

Build enough operating-system, command-line, C, executable-format, and Ghidra literacy to start a small reverse-engineering task with evidence.

## Student Tasks

1. Review the five foundation notes in `notes/`.
2. Practice basic Linux navigation and file inspection commands.
3. Read a small C program and identify variables, branches, loops, functions, strings, and input paths.
4. Explain how C source becomes an executable file.
5. Open a small binary in Ghidra and record the main observations: functions, strings, xrefs, and decompiler output.

## Constraints

- Use `python3` for Python commands.
- Keep evidence reproducible: commands, screenshots, notes, or exported text should identify what was observed.
- Do not store prohibited binaries, credentials, private data, or exploit payloads in this repository.
