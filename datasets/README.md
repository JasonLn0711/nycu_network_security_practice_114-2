# Datasets

Store packet captures, binaries, logs, and sample inputs here when a lab or homework needs reproducible data.

## Rules

- Use lowercase kebab-case filenames.
- Do not store credentials, private data, malware samples, or unsafe exploit payloads.
- Add a short README beside any non-obvious dataset explaining source, purpose, and expected use.
- Link datasets from the relevant lab or homework instead of duplicating files.
