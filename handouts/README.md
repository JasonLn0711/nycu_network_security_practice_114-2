# Handout Archive

This folder is the raw course-material archive for NYCU `Network Security Practice`.

The files here are official or third-party class materials. Keep them separate from personal notes because the repository license does not cover the raw handouts.

## Naming Rule

- Use lowercase kebab-case.
- Preserve the course ordering prefix, such as `a`, `b1`, `c4`, or `d6`.
- Keep original course titles visible in indexes instead of in filenames.
