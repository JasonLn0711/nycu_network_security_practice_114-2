## Course Focus

The syllabus frames the course around security attacks and defense mechanisms, with real-world examples, team projects, mini-report presentations, and demos.

Topic areas listed in the syllabus:

- Introduction
- Vulnerabilities and exploits
- Crypto primitives
- Network security
- Host security
- Code integrity
- Web security
