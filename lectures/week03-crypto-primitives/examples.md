# Week 03 Examples

## Hash Example

If two downloaded files have different hashes, at least one byte differs. A hash alone does not prove who created the file.

## MAC Example

A server can attach a MAC to a message so a receiver with the same secret key can detect tampering and confirm the message came from someone who knows the key.

## Signature Example

A software vendor signs a release. A user verifies the signature with the vendor public key before installing.

## Practice Prompt

For each scenario, choose the right primitive:

- detect accidental file corruption
- protect a message from being read
- prove a software package came from a vendor
- detect tampering by a party without the shared secret
