# Week 04 Key Concepts

## Network Security Lens

Network security asks what communication happened, which protocol fields prove it, and which controls could prevent or detect abuse.

## Core Topics

- TCP/IP layering
- addressing and routing
- DNS lookup and trust boundaries
- reconnaissance vs intrusive testing
- packet capture as evidence
- vulnerability scanning scope and authorization

## Exam Cue

When describing reconnaissance, state the scope and authorization boundary. A defensive lookup is different from intrusive testing.
