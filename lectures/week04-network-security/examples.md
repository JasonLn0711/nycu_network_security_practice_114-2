# Week 04 Examples

## Packet Evidence Example

A TCP connection can be described with source IP, destination IP, source port, destination port, flags, timestamps, and payload clues. The conclusion should name the packets that support it.

## DNS Example

An MX record identifies mail exchangers for a domain. It does not prove those servers are vulnerable.
