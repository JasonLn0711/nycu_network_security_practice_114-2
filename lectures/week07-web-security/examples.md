# Week 07 Examples

## Input Handling Example

If a server builds a database query by concatenating user input, the user may be able to change query structure. Parameterized queries keep data separate from query syntax.

## Session Example

If a session token is stolen, the attacker may act as the user until the token expires or is revoked. Defenses include secure cookies, rotation, monitoring, and re-authentication for sensitive actions.

## Practice Prompt

Pick one web request and identify:

- input fields
- authenticated user state
- server-side action
- data touched
- likely validation point
