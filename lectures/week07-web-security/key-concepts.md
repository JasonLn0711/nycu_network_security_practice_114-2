# Week 07 Key Concepts

## Web Security Lens

Web security studies how untrusted requests interact with server code, data stores, browser behavior, and authentication state.

## Core Concepts

- request and response
- input validation
- authentication and authorization
- session state
- SQL injection
- cross-site scripting
- cross-site request forgery
- server-side request forgery
- file upload risk
- logging and monitoring

## Exam Cue

For a web vulnerability, name the trust boundary: browser to server, user to database query, user content to another browser, or server to internal network.
