# Week 06 Key Concepts

## Code Integrity Lens

Code integrity asks whether the code being loaded or executed is the code that policy intended to trust.

## Core Concepts

- file hash
- digital signature
- trusted publisher
- secure update path
- loader policy
- allowlist and blocklist
- tamper evidence
- rollback risk

## Exam Cue

A hash can detect a change. A signature can also connect the code to a signing identity, assuming the key and trust root are valid.
