# Week 05 Key Concepts

## Host Security Lens

Host security controls who can run code, what code can access, and how privilege is limited after a compromise.

## Core Concepts

- User and group identity
- File permissions and ACLs
- RUID and EUID
- Set-UID programs
- POSIX capabilities
- Least privilege
- DAC and MAC
- Bell-LaPadula confidentiality rules
- SELinux type enforcement
- Windows access tokens and security descriptors

## Exam Cue

For Set-UID questions, identify the privilege transition first. Then list the trusted inputs, untrusted inputs, and safer design choice.
