# Week 01 Examples

## System Behavior Example

A browser opening a file depends on the operating system for file access, memory allocation, and process isolation. A security failure becomes serious when a browser bug escapes that process boundary or reaches higher privilege.

## Practice Prompt

Pick one common application and list:

- what files it reads or writes
- what network connections it might open
- what permissions it should not need
- what evidence would show suspicious behavior

## Review Question

Why is "malware runs as a process" a more useful starting point than "malware is bad software"?
