# Lab: Linux Command-Line Basics

## Goal

Practice file navigation, inspection, permissions, and program execution before reverse-engineering work.

## Tasks

- Use `pwd`, `ls`, and `cd` to map a small directory tree.
- Create, copy, move, and remove practice files.
- Inspect file contents with `cat`, `less`, `head`, and `tail`.
- Compare executable and non-executable permissions with `ls -l` and `chmod`.

## Expected Evidence

- Command transcript or concise command list.
- Notes explaining relative vs absolute paths.
- One permission example before and after `chmod`.
