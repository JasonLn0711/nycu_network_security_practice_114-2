# Lab: Wireshark Analysis

## Goal

Practice packet inspection and turn network observations into clear evidence.

## Tasks

- Open a packet capture or perform a controlled local capture.
- Filter by protocol, host, or conversation.
- Identify packet fields that explain the observed behavior.
- Export or document the key packets.

## Expected Evidence

- Capture source and scope.
- Filters used.
- Packet numbers or screenshots for the key observations.
- Short conclusion connecting packets to protocol behavior.
