# Lab: Ghidra Binary Analysis

## Goal

Practice the first-pass reverse-engineering workflow in Ghidra.

## Tasks

- Create a project and import a small binary.
- Run auto-analysis.
- Identify entry points, functions, strings, and xrefs.
- Compare listing, function graph, and decompiler views.

## Expected Evidence

- Binary name and hash if available.
- Ghidra project notes.
- Three observed functions or strings.
- A short explanation of the most likely input check or program behavior.
