# Lab: DNS Reconnaissance

## Goal

Practice DNS observation in a safe, defensive, and bounded way.

## Tasks

- Query basic DNS records for an allowed domain.
- Compare A, AAAA, CNAME, MX, NS, and TXT records when available.
- Record what each result implies and what it does not prove.
- Note the boundary between passive lookup and intrusive testing.

## Expected Evidence

- Commands used.
- DNS record output.
- Short interpretation of each relevant record type.
- Explicit note that intrusive scanning is outside this lab unless separately authorized.
